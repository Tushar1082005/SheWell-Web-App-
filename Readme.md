🌸 SheWell – AI-Powered Women’s Health & Wellness Platform
SheWell is a cross-platform women’s health and wellness solution designed with compassion and technology. With both a mobile app (React Native) and a web portal, SheWell offers AI-powered support for menstrual health, pregnancy, mental well-being, emergency care, and more — all while protecting privacy and supporting accessibility.

🔥 Key Features
🤖 AI-Powered Health Support
Multilingual chatbot for:

Menstrual health

Pregnancy support

Mental health guidance

AI-based symptom checker

📅 Tracking & Reminders
Period and ovulation tracker

AI-enhanced predictions

Sync with Google Calendar

Personal history stored securely

🧘 Wellness & Safety
Guided meditation

Emergency SOS alert system

Nutrition and lifestyle guidance

Anonymous usage mode

🩺 Medical & Community Care
Free pad and medical aid locator

Book online doctor consultations

Support forums for PCOS, endometriosis

Blockchain-based secure health record storage

🌐 Platforms
📱 React Native Mobile App
Voice-enabled interface

Lightweight and privacy-friendly

Android compatible (iOS coming soon)

Uses Health Connect / Google Fit integration

💻 Web Application
Accessible on all modern browsers

Supports anonymous browsing and forum posting

Admin dashboard and doctor portal (optional)

🧑‍💻 Tech Stack
Layer	Technologies Used
Frontend	React (Web), React Native (App), Tailwind CSS
Backend	Node.js, Express.js
Database	MongoDB (Mongoose)
AI/ML	GPT APIs, Python microservices (for advanced AI)
Voice & NLP	Web Speech API, Expo Speech, Google Translate API
Health APIs	Google Fit / Health Connect
Blockchain	IPFS, Ethereum-based record storage (optional)
